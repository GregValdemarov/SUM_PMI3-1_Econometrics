# Пояснения к models
В файлах данной папки лежат все модели по каждой форме транспортировки + общая (где нет разделения по формам) в порядке исключения самых незначащих компонент на уровне значимости 0,05. <br><br>

Кодировки:
- av -> авиа;
- gen -> общая;
- ra -> ЖД;
- se -> морской;
- tr -> грузовой.
